Javascript Premitive (Value) type

  1. Number
    • Interger: is a numerical value composed of digits that makes a whole number (Can be positive or negative) and does not have any fractional or decimal value.
    • Number: sometimes called a floating point number, is a numeric value, positive or negative, that contains a fractional portion after the decimal point.
    • NOTE:In JS, there is only NUMBER value type that including also interger

  2. String
    • A string is a set of alphanumeric characters, digitsm symbols and special characters represented together

  3. Boolean
    • True
    • False
    • Undefined

  4. Undefined: an empty variable

  5. Null: is used when we want to clear the value of a variable

Variables

  1. Variables: A variable is a programmer defined label of a computer memory location.
    • WE commonly use camelCase to write our variable names
    • The values inside a variable can always be changed, that's why JS is called a dynamic language
    • Variable's names can not be a reserved keyword
    • Should be meaningful
    • Cannot start with a number
    • Cannot contain a space or hyphen

  2. Array: is another data type that allows you to store a simple collection of values inside of a single variable
    • Arrays are defined by square brackets [kinda looks like a books self]
    • Arrays use zere-based indexing, first value is indexed Zero
    • Example: console.log("The first name is: ", staffNames[0])

  3. Object: is another data type that allows you to store a complex collection of key-value pairs inside of a single variable.
    • Objects are defined by curly brackets { }
    • Example:
      let myObject = {
           id: 391,
           name: "Bob",
      }
      console.log("The name is: ", myObject.name)
    • An array can contain objects and vice-versa
    • Example:
      let students = [
      { id:234,
           name: "Albert",
           course: ["Biology","Math","English"]
      },
      { id:236,
           name: "Sara",
           course: ["English", "Art"],
      },
      • ]
      console.log("The secon student in the list is: ",students[1].name)
    • NOTE:
        Dotnotation: (student.name) is used more because it shorter
        Bracketnotation: (student['name']) is used incase the selection is dynamic
        example:let selection = 'name'
                 student[selection] = Mary

Conditionals

  1. Switch: Multiple if...else statements

  2. Ternary: Single if...else statement
    • Example:
      let canDrive = age > 16? 'yes' : 'no'

  3. Loops: to perform a task more than once without writing duplicate code

  4. while: to do a loop while a certain condition is true, or until a certain condition is no longer true
